**Задание:**

Роботы всюду, и даже тут. Они скрывают флаг.

---
**Решение:**

Смотрим файл `robots.txt` платформы и находим там флаг:

```text
User-agent: *
Disallow: /admin

# Флаг для задачи «Роботы»:
# BugCTF{XXXXXXXXXXXXXXX}
```
Файл `robots.txt` используется для предоставления веб-роботам, например поисковым роботам, инструкций о том, какие разделы сайта разрешено или запрещено сканировать и индексировать.

Файл `robots.txt` располагается в корневой директории вашего веб-сайта на веб-сервере.
